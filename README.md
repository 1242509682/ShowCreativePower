# ShowCreativePower 显示旅途力量
- 作者: 羽学、西江小子
- 出处: TShock官方群816771079
- 这是一个Tshock服务器插件，主要用于：在非旅途模式下对配置名单内的玩家显示力量菜单，并支持手游。

## 更新日志

```
v1.0.1
移除无用代码,修复手游玩家的力量菜单偶尔消失bug
加入了物品排除表功能对应指令(/scp i 与 /scp is)
修改排除表指令/scp is <物品名>
存在则移除,不在则添加,触发则刷新,支持-i检测手持物品
加入每行个数配置项（控制名单和排除物品表每行显示数）
将进服开关指令:/scp at改为/scp j
移除add与del指令,改为/scp md （存在则移除,不在则添加）
加入了:非名单玩家使用/scp指令只会显示名单,并提示自己不在名单内

v1.0.0
在Hufang的ShowPowerMenu基础上进行加强
支持PE显示力量菜单（由西江小子贡献代码）
优化了移除名单后不再踢出玩家
目前存在BUG:
手游玩家的力量菜单偶尔会消失,需要重新使用指令/scp on 开启
```

## 指令

| 语法                             | 别名  |       权限       |                   说明                   |
| -------------------------------- | :---: | :--------------: | :--------------------------------------: |
| /scp  | 无 |   spm.use    |    指令菜单    |
| /scp on | 无 |   spm.use    |    开启自己的力量菜单    |
| /scp off | 无 |   spm.use    |    关闭自己的力量菜单    |
| /scp list | ls |   spm.use    |    列出力量名单    |
| /scp item | i |   spm.use    |    列出研究排除物品表    |
| /scp items | is |   spm.admin    |    修改研究排除物品表    |
| /scp pe | 无 |   spm.admin    |    切换手游是否显示    |
| /scp join | j |   spm.admin    |    切换进服开关    |
| /scp md <玩家名> | 无 |   spm.admin    |    修改玩家名单    |
| /reload  | 无 |   tshock.cfg.reload    |    重载配置文件    |

## 配置
> 配置文件位置：tshock/显示旅途力量.json
```json
{
  "插件开关": true,
  "允许手游显示力量菜单": true,
  "解锁全物品研究": true,
  "进服自动显示菜单": true,
  "名单与物品每行显示数量": 10,
  "旅途力量名单": [
    "羽学",
    "西江",
    "灵乐",
    "安安"
  ],
  "研究物品排除表": [
    74,
    73,
    72,
    71
  ]
}
```
